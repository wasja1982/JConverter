2) Для работы необходима установленная Java-машина.

3) Пример запуска:
3.1) минимальный (для запуска из каталога бекапа, требуемые файлы лежат в каталоге "_s1", таблицы без префикса):
[code]@java -jar "JConverter.jar"[/code]
3.2) с одним параметром (передается путь к каталогу бекапа, требуемые файлы лежат в каталоге "_s1", таблицы без префикса):
[code]@java -jar "JConverter.jar" ucozDUMP[/code]
3.3) с двумя параметрами (первый параметр - путь к каталогу бекапа, второй - название каталога с требуемыми файлами, таблицы без префикса):
[code]@java -jar "JConverter.jar" ucozDUMP[/code]
3.4) полный вариант с тремя параметрами (первый параметр - путь к каталогу бекапа, второй - название каталога с требуемыми файлами, третий - префикс таблиц):
[code]@java -jar "JConverter.jar" ucozDUMP _s1 fps_[/code]
Если путь содержит пробелы, то его необходимо взять в кавычки. Пути указываются без завершающего слеша.
